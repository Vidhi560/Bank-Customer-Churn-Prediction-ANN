# Bank Customer Churn Prediction using Artificial Neural Network (ANN)

## Overview

This project focuses on predicting whether a bank customer is likely to leave the bank using an Artificial Neural Network (ANN). The model is built using TensorFlow and Keras to perform binary classification based on customer demographic and financial information.

The project demonstrates the complete machine learning workflow, including data preprocessing, feature engineering, model development, training, evaluation, and prediction.

---

## Problem Statement

Customer churn prediction helps banks identify customers who are at risk of leaving, allowing them to improve customer retention strategies and business decision-making.

---

## Features Used

- Credit Score
- Geography
- Gender
- Age
- Tenure
- Balance
- Number of Products
- Has Credit Card
- Is Active Member
- Estimated Salary

### Target Variable

- **0** → Customer did NOT exit the bank
- **1** → Customer exited the bank

## ANN Architecture

- Input Layer
- Hidden Layer 1 (ReLU)
- Hidden Layer 2 (ReLU)
- Hidden Layer 3 (ReLU)
- Output Layer (Sigmoid)

Optimizer: Adam

Loss Function: Binary Crossentropy

Evaluation Metric: Accuracy

---

## Model Performance

- Model Accuracy: **85–86%**
- Binary Classification Model
- Efficient prediction of customer churn

Prediction Labels:

- **0** → Customer Retained (No Churn)
- **1** → Customer Churned
